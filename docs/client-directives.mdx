## `@skip`

:::note GraphQL Specification

The `@skip` directive is a specification specific directive and must work as
defined in the GraphQL Specs.

:::

## `@include`

:::note GraphQL Specification

The `@include` directive is a specification specific directive and must work as
defined in the GraphQL Specs.

:::

:::tip Opposite Actions

The `@include` and `@skip` directives operate in opposite actions to each other.
Using these directives together on the same fields will produce problematic
results.

:::
